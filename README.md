## Overview
TravelBuddy is a web application built on Streamlit that helps users plan their travel itineraries efficiently. Whether you're organizing a solo adventure or a group trip, TravelBuddy simplifies the process of creating and managing travel plans.

## Project Setup
1. Clone the repository: `https://github.com/ManasPatni/Travelling-agent_ai-chatbot-`
2. Navigate to the project directory: `cd travelbuddy`
3. Install dependencies: `pip install -r requirements.txt`
4. Run the application: `streamlit run app.py`

## Usage
Once the application is running, open your web browser and go to `https://travelbuddy07.streamlit.app/` to access TravelBuddy. Use the intuitive interface to:
- Enter destination details and travel dates.
- Add activities, attractions, and notes to your itinerary.
- View and edit your itinerary dynamically.

## Key Features
- Interactive dashboard for managing travel plans.
- Allows users to set and track travel budgets
- Provides real-time weather updates for selected destinations to help users plan accordingly
